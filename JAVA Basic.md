# JAVA Basic.
## 변수.

`int a;`

`String b;`

## 변수명.

1. 숫자로 시작할 수 없다.
2. `_`, `$` 이외의 특수문자는 사용할 수 없다.
3. JAVA의 키워드는 사용할 수 없다.

## 자료형.

1. int : 정수.
2. String : 문자열.
3. long
4. double
5. boolean
6. char
7. StringBuffer
8. List
9. Map

## 사용자 정의 자료형.

`class Animal {
}`에서 `Animal cat;`으로 Class에 맞는 자료형을 정의하여 사용할 수 있음.
